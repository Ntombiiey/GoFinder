# GoFinder

**GoFinder** is a web-based platform designed to help users **discover local businesses** such as restaurants, cafes, and social spots. Users can sign up, log in, and explore locations with ease.

## Features
- User registration and login system
- Password recovery (forgot/reset)
- Browse and search local businesses
- Responsive and clean user interface

## Technologies Used
- **Frontend:** HTML, CSS
- **Backend:** PHP
- **Database:** MySQL
